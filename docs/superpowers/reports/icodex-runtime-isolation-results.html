Требования по разделам

РазделТребование
DecisionsPer-project дома, централизованно под $ICODEX_ROOT/.codex-homes/<id>/; целевые репо чистые.
DecisionsShared auth.json через symlink — один логин; изолируются sessions/history/memory/logs.
DecisionsSandbox по умолчанию workspace-write (safe by default).
DecisionsEscalation до danger-full-access явный: флаг --full-access или ICODEX_SANDBOX + stderr warning.
Decisionsapproval_policy неизменна (on-request) — escalation не bypass'ит approvals.
DecisionsDependency-light: только bash / awk / ln, ноль новых зависимостей.
Architecture · Path splitРасщепить ICODEX_HOME_DIR на ICODEX_SHARED_DIR (assets) и динамический per-project ICODEX_HOME_DIR (state).
Architecture · Project identity<id> = <basename>-<short-sha256>; корень = git-toplevel или realpath cwd.
Architecture · Module changesПравки: init.sh, isolated.sh, install.sh, superpowers.sh, новый sandbox.sh, args.sh, icodex.sh.
Architecture · Sandbox escalationPrecedence default < ICODEX_SANDBOX < --full-access; валидация значений; идемпотентный awk-upsert sandbox_mode.
Architecture · Config generationPer-project config.toml: copy template + idemпotentный upsert managed-строк (sandbox, marketplace, binary grant, trust).
MigrationShared assets остаются в .codex-isolated; осиротевший runtime безопасен (CODEX_HOME → .codex-homes); .codex-homes/ в gitignore.
Testing (TDD)Расширить/добавить: test_isolated, новый test_sandbox, test_install/test_update_scope, test_plugin, test_smoke, trust-тест.
Out of scopeGC домов, --list-homes, isolated-auth, 3-уровневый sandbox UX — future.

Success Criteria (Verification)

#Критерий
1Запуск из чужого репо → CODEX_HOME под .codex-homes/<id>/, отдельный на проект; сессии там, не в .codex-isolated.
2Два разных репо → разные дома; нет cross-project утечки session/memory.
3Дефолт-запуск → sandbox_mode = workspace-write; без warning.
4--full-access / ICODEX_SANDBOX=danger-full-access → danger-full-access в config + stderr warning.
5install/update пишут только под .codex-isolated/bin, дом не создают.
6Все обновлённые и новые тесты проходят.
