Проверка intent — 2026-07-08

Интент фиксирует цель: добавить opt-in телеметрию для двух поверхностей наблюдаемости. Grafana отвечает за ops-метрики сессий, Langfuse отвечает за LLM traces с привязкой к проекту и сессии.

Проверка подтверждает, что документ содержит все обязательные разделы IDD, outcomes наблюдаемы, hard constraints отделены от steering constraints, autonomy zones покрывают четыре зоны, а stop rules имеют измеримый done-when.

Проверка spec — 2026-07-08

Спека фиксирует hybrid architecture: Codex OTel даёт metadata-only ops telemetry для Grafana, а отдельный local trusted capture layer даёт full-fidelity traces для Langfuse.

Проверка подтверждает покрытие intent outcomes: telemetry выключена по умолчанию, config surface с одним mode-switch сохранён, Grafana не получает prompt/response bodies, Langfuse full capture разрешён только для локального trusted endpoint.
